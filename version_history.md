# Version History & Roadmap

## v1.0.0 - Initial Release (09 June 2026)

### Added

* Telemetry CSV ingestion
* Speed, throttle, brake and steering analysis
* Acceleration calculation
* Event detection
* Track map visualization
* Driver comparison dashboard
* Sample telemetry datasets

---

## v1.1.0 - Corner Analysis (Planned)

### Planned Features

* Corner detection
* Corner entry speed
* Apex speed calculation
* Corner exit speed
* Corner-by-corner driver comparison

### Goal

Provide basic race engineering insights instead of only displaying telemetry.

---

## v1.2.0 - Sector Analysis (Planned)

### Planned Features

* Sector splitting
* Sector times
* Fastest sector identification
* Driver sector comparison

### Goal

Identify where lap time is gained or lost.

---

## v2.0.0 - Driver Performance Analysis (Planned)

### Planned Features

* Speed delta analysis
* Braking point comparison
* Throttle application comparison
* Automated driver feedback

### Goal

Turn the dashboard into a driver coaching tool.

---

## v3.0.0 - Racing Line Analysis (Future)

### Planned Features

* Racing line visualization
* Speed heatmaps
* Optimal line estimation
* GPS trajectory comparison

### Goal

Introduce vehicle dynamics and racing line optimization concepts.
